Using accident and casualty datasets, I built an interactive dashboard to analyze road accident trends and identify critical factors affecting road safety.
🔍 Key Insights Covered:
✔ Casualties by Accident Severity
 ✔ Urban vs Rural Accident Distribution
 ✔ Daylight vs Darkness Accident Analysis
 ✔ Road Surface Condition Impact
 ✔ Casualties by Road Type
 ✔ Monthly Casualty Trend Analysis (CY vs PY)
 ✔ Interactive Filters & Drill-down Analysis
🛠 Tools & Skills Used:
🔹 Microsoft Power BI
 🔹 Power Query for Data Cleaning & Transformation
 🔹 Data Modeling
 🔹 DAX Measures
 🔹 Data Visualization & Storytelling
 🔹 Interactive Slicers & KPI Design

 ✅ Trend analysis

 ✅ User-friendly and interactive UI
